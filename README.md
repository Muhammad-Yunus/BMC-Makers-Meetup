# BMC-Makers-Meetup
### Buitenzorg Makers Club - Makers Meetup - Code &amp; Files

> BMC Makers Meetup is a weekly gathering of makers, hobbyists, and tech enthusiasts in Buitenzorg (Bogor), Indonesia. The meetup provides a platform for members to share their projects, collaborate on ideas, and learn new skills related to DIY electronics, programming, 3D printing, robotics, and more.
